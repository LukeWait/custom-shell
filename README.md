## Installation
#### Windows
Not currently supported - maybe in the future.

#### Linux
1. Clone the repository:
    ```sh
    git clone https://github.com/LukeWait/custom-shell.git
    cd custom-shell
    ```

2. Make executable and run the setup script:
    ```sh
    chmod -x setup.sh
    ./setup.sh
    ```
